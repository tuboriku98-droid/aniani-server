## はじめに
本プロジェクトは **HTML / CSS** を用いて作成した、  
アニメ作品を紹介する **静的Webサイト** です。

Web制作の基礎学習に加え、  
**Linux環境でのサーバ構築・運用の理解を深めること** を目的として制作しました。

## 使用技術

### フロントエンド
- HTML
- CSS

### サーバ / インフラ
- Docker
- Ubuntu
- Apache2
- chmod / chown（権限管理）
- ログ分析（アクセスログ / エラーログ）

---

## サーバ構築手順

1. **Docker** を用いて Linux（Ubuntu）環境を作成  
2. **Apache2** をインストール  
   - Webサーバとして HTML コンテンツを配信  
3. HTML / CSS を **nano** で編集  
   - ブラウザで表示確認  
4. 画像ファイルを **WebP形式** に変換  
   - ページ表示速度を意識した軽量化  
5. Ubuntu 全体および nano に **日本語ロケール** を追加  
   - 運用時の作業効率向上  
6. **エラーログ・アクセスログ** の確認  
   - トラブル発生時の原因特定  

---

## トラブルと解決内容

### Ubuntuで日本語入力時に文字化けが発生
**対応:**  
日本語ロケールをインストールし設定を永続化。  
コンテナ再起動後も日本語設定が維持されるよう対応しました。

### 特定の画像が表示されない
**対応:**  
画像ファイルの権限を統一し、  
ファイル名を修正することで正常に表示されるようになりました。

## 製作時に工夫した点（セキュリティ・運用）

- `/images` などのディレクトリを直接検索しても  
  **ファイル一覧が表示されないよう設定**  
  - サイト表示には影響なし  

- エラーページに **Apache や OS のバージョン情報を非表示**  
  - 不要な情報公開を防ぎ、攻撃リスク低減を意識  

- **HTTPメソッドを制限（Apache設定）**  
  - 許可: `GET`, `POST`  
  - 拒否: `PUT`, `DELETE`, `OPTIONS`
