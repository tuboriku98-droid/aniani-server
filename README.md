＃＃ サーバ構築手順

1,DockerでLinux環境を作成

2,Apacheをインストール
-webサーバとしてHTMLコンテンツ配信

3,HTML,CSSをnanoに読み込ませる
-ブラウザで表示確認

4,画像ファイルをwebpにして軽量化
-ページ表示速度意識し軽量化

5,Ubuntu全体とnanoに日本語ロケールを追加
-運用時の作業効率化

6,エラーログ、アクセスログの確認

＃＃＃ トラブルと解決内容

トラブル:Ubuntuが日本語を打つと文字化けしてしまう

対応:日本語ロケールをインストール→永続化、コンテナ再起動後も日本語設定が維持されるようにしました。

トラブル:特定の画像が表示されない

対応:全ての画像の権限を統一、ファイル名修正をしたら表示されるようになりました。
